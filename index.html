Question-2: What is the difference between innerHTML, innerText, and textContent ?

textContents is all text contained by an element and all its children that are for formatting purposes only. innerText returns all text contained by an element and all its child elements. innerHtml returns all text, including html tags, that is contained by an element.

Question-3: What is event delegation in the DOM?

In DOM event delegation, a single event listener is attached to a parent element to handle events from its child elements, leveraging event bubbling (where events propagate up the DOM tree) for efficient event handling.

Question-4: What is event bubbling in the DOM?

Event bubbling can be defined as a phase of Event propagation in which if an event occurs on a particular element will propagate or bubble up to the ancestor or the parent elements in the DOM hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, you can manipulate the DOM by creating, adding, and removing elements. To create an element, use a method that generates a new HTML tag. To add it, append it to a parent element or insert it before an existing one. To remove an element, either delete it directly or remove it from its parent. These actions allow dynamic updates to web pages.
